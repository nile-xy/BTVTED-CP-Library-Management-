How to run the System

Install the latest version of Xampp
Clone the repository or simply download the codes.
Extract the codes and move it to the XAMPP folder inside the htdocs folder.
Enable the Apache and MySQL functions in XAMPP and access localhost via phpmyadmin
create a new database. It should have the same name as the database on the folder.
After creating click import and upload the local database.
Click import and you are good.

Lastly to access the system go to your browser type localhost/foldername
